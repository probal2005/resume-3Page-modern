# 📄 Resume – 3 Page Modern

A modern, responsive, and professional **3-page resume website** built using **HTML, CSS, and JavaScript**. Designed with a clean UI, structured layout, and optimized for both screen viewing and print/PDF export.

---

## 🚀 Features

* ✅ Modern and clean user interface
* ✅ Fully responsive (mobile + desktop)
* ✅ Print-ready (export as PDF بسهولة)
* ✅ Organized sections for professional presentation
* ✅ Social and coding profile links
* ✅ Skill bars with visual representation
* ✅ Font Awesome icons integration
* ✅ Smooth hover effects

---

## 📂 Sections Included

* 🔹 Header (Profile + Contact Information)
* 🔹 Social Links (LinkedIn, GitHub, etc.)
* 🔹 Summary
* 🔹 Education
* 🔹 Projects
* 🔹 Skills & Languages
* 🔹 Programming Languages
* 🔹 Certifications
* 🔹 Achievements
* 🔹 Hobbies

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3 (Flexbox + Grid + Media Queries)**
* **JavaScript (Print Function)**
* **Google Fonts (Poppins)**
* **Font Awesome Icons**

---

## 🖥️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/probal2005/resume-3Page-modern.git
   ```

2. Open the folder

3. Run the project:

   * Open `index.html` in your browser

---

## 🖨️ Export as PDF

* Click on **"Print Resume"** button
* Choose **Save as PDF**

---

## 🎨 Customization

You can easily modify:

* 👤 Profile image → `assets/profile.jpg`
* 🧾 Personal details → inside `index.html`
* 🎨 Colors → edit CSS styles
* 📊 Skills → adjust percentage bars
* 🔗 Social links → replace URLs
